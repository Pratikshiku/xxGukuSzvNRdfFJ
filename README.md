## City Introduction Tourism Introduction System Inner Mongolia

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/11f590b0c57848fdb3b8dbc5c6472042/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `Visual Studio Code(VScode)`
- `IDEA`
- `Eclipse`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Tourist, Administrator.

**Tourist Functions:**
1. Login
2. Register
3. Homepage
4. City Introduction
5. Featured Attractions
6. Featured Cuisine
7. Historical Culture
8. Message Center
9. Secure Logout


**Administrator Functions:**
1. Login
2. City Management
3. Featured Attractions Management
4. Featured Cuisine Management
5. Message Management
6. Carousel Management
7. Secure Logout
